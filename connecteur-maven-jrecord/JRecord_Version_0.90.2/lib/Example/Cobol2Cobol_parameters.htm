Parameters for Cobol2Cobol

                                           
Program arguments:
          
    -I  or -InputFile    : Input file
    -InputCopybook       : Cobol/Xml Copybook file
    -InputFileStructure  : Input File Structure (or File-Organisation):
        Default          : Determine by  
        Text             : Use Standard Text IO              
        Fixed_Length     : Fixed record Length binary 
        Binary           : Binary File, length based on record 
        Mainframe_VB     : Mainframe VB File 
        Mainframe_VB_As_RECFMU    : Mainframe VB File including BDW (block descriptor word)
        FUJITSU_VB       : Fujitsu Cobol VB File 
        Open_Cobol_VB    : Gnu Cobol VB File 
    -InputCharacterSet   : Input font or character set
    -InputDialect        : Cobol Dialect
        0 or Intel       : Intel little endian 
        1 or Mainframe   : Mainframe big endian (Default) 
        4 or Open_Cobol_Little_Endian_(Intel)    : Gnu:Cobol 
        2 or Fujitsu     : Fujitsu Cobol 

    -O  or -OutputFile   : Output file
    -OutputCharacterSet  : Output font or character set
    -OutputCopybook      : Output Cobol/Xml Copybook file
    -OutputFileStructure : Output File Structure (or File-Organisation)
            See: -InputFileStructure for details
    -OutputCharacterSet  : Output font or character set
    -OutputDialect       : Output Cobol Dialect) 
            See: -InputDialect for details
