Introduction

JRecord provides

  1. IO routines for Length Based (Lines are either fixed length or a length field at the start f the record) files.
  2. IO routines for Cobol files (both Text and Binary).

JRecord was split off as a separate project from the RecordEditor project on the request of several users.

The Example Directory holds:

  • Sample bat files / shell scripts for running java program XmplLineIOD with JRecord to read a Cobol binary file.
    The program XmplLineIOD.java
    • can be found in the Examples Directory.
  • Sample bat files / shell scripts for running Cobol2Csv.jar, Csv2Cobol.jar and Cobol2Cobol.jar.

Jars in this directory

  • cb2xml.jar - Cobol analysis Jar
  • JRecord.jar - JRecord library
  • Cobol2Cobol.jar - Copy from one Cobol Data file to another (matching the data names).
  • Cobol2Xml.jar - Copy a single record Cobol Data file to/from Xml file.

Warning - Cobol2Csv, Csv2Cobol and Cobol2Cobol are example programs I would suggest testing throughly before using them in production.

Note: All three utilities now accept Xml-Schema's instead of Cobol-Copybooks

Parameters for Cobol2Cobol

                                           
Program arguments:
          
    -I  or -InputFile    : Input file
    -InputCopybook       : Cobol/Xml Copybook file
    -InputFileStructure  : Input File Structure (or File-Organisation):
        Default          : Determine by  
        Text             : Use Standard Text IO              
        Fixed_Length     : Fixed record Length binary 
        Binary           : Binary File, length based on record 
        Mainframe_VB     : Mainframe VB File 
        Mainframe_VB_As_RECFMU    : Mainframe VB File including BDW (block descriptor word)
        FUJITSU_VB       : Fujitsu Cobol VB File 
        Open_Cobol_VB    : Gnu Cobol VB File 
    -InputCharacterSet   : Input font or character set
    -InputDialect        : Cobol Dialect
        0 or Intel       : Intel little endian 
        1 or Mainframe   : Mainframe big endian (Default) 
        4 or Open_Cobol_Little_Endian_(Intel)    : Gnu:Cobol 
        2 or Fujitsu     : Fujitsu Cobol 

    -O  or -OutputFile   : Output file
    -OutputCharacterSet  : Output font or character set
    -OutputCopybook      : Output Cobol/Xml Copybook file
    -OutputFileStructure : Output File Structure (or File-Organisation)
            See: -InputFileStructure for details
    -OutputCharacterSet  : Output font or character set
    -OutputDialect       : Output Cobol Dialect) 
            See: -InputDialect for details
