JRecord Compare Programs

The programs in this directory provide a simple way to check a new version of JRecord against the previous version of JRecord.

There are 2 basic programs:

WriteJRecordCompareFile.bat This program reads through all the Cobol-Copybooks in a supplied directory (cobol_copybooks) and writes the copybooks record and field details (position, length, field-type and decimal etc) to a gzipped text 'Layout' file (JRecordCompare.txt.gz).

This 'Layout' file (JRecordCompare.txt.gz) provides a "baseline" that can be used to compare with future version of JRecord.

RunJRecordCompare.bat This program reads through all the Cobol-Copybooks in a supplied directory (cobol_copybooks) and compares the Record, Field details (position, length, field-type and decimal etc) against the "baseline" Record / Field details store in the 'Layout' file (JRecordCompare.txt.gz).

I suggest you create your own Cobol directories, bat/shell scripts and 'baseline' files to test future versions of JRecord. The more eyes the better.
